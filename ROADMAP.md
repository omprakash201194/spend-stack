# SpendStack Roadmap

This document outlines the planned evolution of SpendStack.

SpendStack is being developed incrementally with a focus on
**privacy-first personal finance management**.

------------------------------------------------------------------------

## Current Focus

The current phase focuses on building the **core foundation of the
application**.

Key areas:

-   Project architecture
-   Identity and workspace model
-   Statement import pipeline
-   Transaction normalization
-   Audit trail system

------------------------------------------------------------------------

## Version Roadmap

### v1.0 -- Core Finance Engine

The first milestone focuses on enabling reliable financial data
ingestion and management.

Planned features:

-   Multi-user profiles
-   Family workspace
-   Statement import system
-   Parser framework
-   Duplicate detection
-   Review queue
-   Normalized transaction model
-   Audit events
-   Local-first database (SQLite)

Goal:

Provide a stable and trustworthy personal finance record system.

------------------------------------------------------------------------

### v1.1 -- Budgeting and Export

Planned capabilities:

-   Budget categories
-   Monthly spending summaries
-   Budget alerts
-   CSV / XLSX exports
-   Basic financial reports

Goal:

Enable users to understand and control spending patterns.

------------------------------------------------------------------------

### v1.2 -- Localization

Features:

-   Multi-language support
-   Currency formatting
-   Region-aware number formatting
-   UI translation support

Goal:

Allow SpendStack to be usable across multiple regions.

------------------------------------------------------------------------

### v1.5 -- Optional Cloud Sync

Cloud sync will be **optional** and privacy aware.

Possible capabilities:

-   encrypted backup
-   multi-device sync
-   conflict resolution
-   user-controlled cloud providers

Goal:

Allow convenience without sacrificing privacy.

------------------------------------------------------------------------

## Future Vision

Potential future capabilities:

-   mobile companion apps
-   financial planning tools
-   recurring transaction detection
-   predictive insights
-   AI-assisted financial summaries
-   bank integrations (optional)

All features will maintain SpendStack's **local-first philosophy**.

------------------------------------------------------------------------

## Principles Guiding Development

SpendStack development is guided by these principles:

-   privacy-first design
-   user ownership of data
-   transparency of financial processing
-   modular architecture
-   explainable financial insights

------------------------------------------------------------------------

## Contribution to Roadmap

The roadmap will evolve based on:

-   user feedback
-   contributor input
-   technical discoveries during development

Suggestions and discussion are welcome through GitHub Issues.
